# 🔗 URLite - URL Shortener

A **MERN-style URL Shortener Web App** that helps users convert long links into short, clean, and shareable URLs.  

Built with **MongoDB, Express, React, and Node.js**, this project makes link management simple, fast, and user-friendly 🚀

---

## 🌟 Features

✨ **Smart URL Shortening**  
🔗 Convert long URLs into short, easy-to-share links  

🎯 **Custom Short Codes**  
✍️ Create personalized short URLs  

⚡ **Instant Redirection**  
🌍 Automatically redirect users to the original link  

📊 **Click Analytics**  
📈 Track how many times your link is used  

🕒 **Recent URLs**  
🗂️ View previously created short links stored in MongoDB  

🎨 **Modern UI**  
💻 Clean, responsive interface built with React
